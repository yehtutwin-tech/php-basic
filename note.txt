isset()
- Returns TRUE if a variable is declared and not null
- http://localhost/dsi/php1/basic/isset_empty.php

empty()
- Returns TURE if a variable is not declared, false, null, ""
- https://www.w3schools.com/php/func_var_empty.asp

Superglobal
$_GET
  - Data is appended to the url
  - Not secure
  - Character limit
  - Bookmark is possible with values
  - GET requests can be cached
  - Better for a search page

$_POST
  - Data is packed inside the body of the HTTP request
  - More Secure
  - No data limit
  - Cannot bookmark
  - POST requests are not cached
  - Better for submitting credentials
